DB Backend Index

Academic year and semester - /v2/academic

Academic Year

  • GET /v2/academic/years - Get all academic years
  • PUT /v2/academic/years - Insert a new academic year
  • GET /v2/academic/years/:year - Get the specified academic year
  • DELETE /v2/academic/years/:year - Delete the specified academic year

Academic Semester

  • GET /v2/academic/years/:year/semesters - Get all semesters of the specified academic year
  • PUT /v2/academic/years/:year/semesters - Insert a new semester for the specified academic year
  • GET /v2/academic/years/:year/semesters/:semester - Get the specified semester
  • PATCH /v2/academic/years/:year/semesters/:semester - Update the specified semester`
  • DELETE /v2/academic/years/:year/semesters/:semester - Delete the specified semester

Subject and Sections - /v2/subjects

Subject

  • GET /v2/subjects - Get all subjects
  • PUT /v2/subjects - Insert a new subject
  • GET /v2/subjects/:subid - Get the specified subject
  • PATCH /v2/subjects/:subid - Update the specified subject
  • DELETE /v2/subjects/:subid - Delete the specified subject

Class (subject in a certain year)

  • GET /v2/subjects/:subid/:year/:semester - Get the class corresponding to the subject in the specified semester
  • PUT /v2/subjects/:subid/:year/:semester - Insert a class corresponding to the subject in the specified semester
  • PATCH /v2/subjects/:subid/:year/:semester - Update the class corresponding to the subject in the specified semester
  • DELETE /v2/subjects/:subid/:year/:semester - Delete the class corresponding to the subject in the specified semester

Section

  • GET /v2/subjects/:subid/:year/:semester/sections - Get all sections of a subject
  • PUT /v2/subjects/:subid/:year/:semester/sections - Insert a new section for the specified subject
  • GET /v2/subjects/:subid/:year/:semester/sections/:sectno - Get the specified section
  • PATCH /v2/subjects/:subid/:year/:semester/sections/:sectno - Update the specified section
  • DELETE /v2/subjects/:subid/:year/:semester/sections/:sectno - Delete the specified section from the subject
  • GET /v2/subjects/:subid/:year/:semester/sections/:sectno/students - Get the list of students in the section

Teachers - /v2/teachers

Teacher

  • GET /v2/teachers - Get all teachers
  • PUT /v2/teachers - Insert a new teacher
  • GET /v2/teachers/:tid - Get the specified teacher
  • PATCH /v2/teachers/:tid - Update the specified teacher
  • DELETE /v2/teachers/:tid - Delete the specified teacher
  • GET /v2/teachers/:tid/advisees - Get the list of students that the teacher advises

Teacher Schedule

  • GET /v2/teachers/:tid/schedules/teach - Get all teaching schedules (in all years) of the specified teacher
  • GET /v2/teachers/:tid/schedules/teach/:year - Get all teaching schedules (in a certain year) of the specified teacher
  • GET /v2/teachers/:tid/schedules/teach/:year/:semester - Get a single teaching schedule of the specified teacher
