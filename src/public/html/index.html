DB Backend Index

Academic year and semester - /v2/academic

Academic Year

  • GET /v2/academic/years - Get all academic years
  • PUT /v2/academic/years - Insert a new academic year
  • GET /v2/academic/years/:year - Get the specified academic year
  • DELETE /v2/academic/years/:year - Delete the specified academic year

Academic Semester

  • GET /v2/academic/years/:year/semesters - Get all semesters of the specified academic year
  • PUT /v2/academic/years/:year/semesters - Insert a new semester for the specified academic year
  • GET /v2/academic/years/:year/semesters/:semester - Get the specified semester
  • PATCH /v2/academic/years/:year/semesters/:semester - Update the specified semester`
  • DELETE /v2/academic/years/:year/semesters/:semester - Delete the specified semester

Companies and Branches - /v2/companies

Company

  • GET /v2/companies - Get all companies
  • PUT /v2/companies - Insert a new company
  • GET /v2/companies/:compid - Get the specified company
  • PATCH /v2/companies/:compid - Update the specified company
  • DELETE /v2/companies/:compid - Delete the specified company

Branch

  • GET /v2/companies/:compid/branches - Get all branches of the specified company
  • PUT /v2/companies/:compid/branches - Insert a new branch for the specified company
  • GET /v2/companies/:compid/branches/:brid - Get the specified branch
  • PATCH /v2/companies/:compid/branches/:brid - Update the specified branch
  • DELETE /v2/companies/:compid/branches/:brid - Delete the specified branch
  • GET /v2/companies/:compid/reviews - Get reviews associated with the specified company

  • GET /v2/faculties/:fid - Get the specified faculty

Departments

  • GET /v2/faculties/:fid/departments - Get all departments that belong to the specified faculty
  • PUT /v2/faculties/:fid/departments - Insert a new department for the specified faculty
  • GET /v2/faculties/:fid/departments/:depid - Get the specified department
  • PATCH /v2/faculties/:fid/departments/:depid - Update the specified department
  • DELETE /v2/faculties/:fid/departments/:depid - Delete the specified department

Head of Department

  • GET /v2/faculties/:fid/departments/:depid/head - Get the head of department of the specified department
  • PUT /v2/faculties/:fid/departments/:depid/head - Insert a new head of department for the specified department
  • PATCH /v2/faculties/:fid/departments/:depid/head - Update the head of department of the specified department
  • DELETE /v2/faculties/:fid/departments/:depid/head - Delete the head of department of the specified department
